- Features -
Function 01
ガラスの表情

一面一面をはんだ付けした素朴でレトロなガラスセードから、やさしくて懐かしい光が広がります。テイストの異なるガラス表面の凹凸が独特の光の模様を映し出します。
   
Function 02
ハンドメイド

いくつもの三角形を交互にハンダ付けして多角形を形状させます。ひとつひとつハンドメイドですので、商品ごとの微妙な大きさの違いや溶接部の擦れなど手作りのテイストをお楽しみください。
   
Function 03
アンティーク

アンティークゴールドのソケットとブラウンの縒りコードを採用することで、よりレトロな佇まいになりました。長さの調節もでき、余分なコードはシーリングカップの中にスッキリと収納できます。
 
 
Function 04
サイズ感

デザインを損なわず実用性を両立させた、ベストなサイズ感を追求。

- Brand information -
 
「人が過ごす場所と時間を心地よく」

インターフォルムは1979年に照明器具のメーカーとして創業して以来、時計やインテリア雑貨にもフィールドを広げ、みなさまの日々の暮らしのひとときを心地よくするお手伝いをしたいという想いを込めて、オリジナルデザインの商品をご提案してきました。

インターフォルムという社名には、「世界に通用するデザインを」という意味が込められています。
そんな理想を持ち、世界のトレンドと、日本の緻密なデザインワークをかけ合わせた商品づくりを続けています。
